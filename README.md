## Create a new repository on the command line

<code>
git init
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/codevea/repo.git
git push -u origin main
</code>

## Push an existing repository from the command line

<code>
git remote add origin https://github.com/codevea/repo.git
git branch -M main
git push -u origin main
</code>
